#RedisManage 
##一、项目简介
-因RedisDesktopManager在服务器上打不开无法使用，临时弄了个基于.Net6的Winform界面程序进行操作，有不对的地方请大家指正，谢谢！
-本项目是基于.Net6 的，Redis库的连接在配置文件App.config中设置修改
##二、安装依赖包
-StackExchange.Redis v2.7.33
-Newtonsoft.Json v13.0.3
##三、项目运行
--按正常winform的几种启动运行模式即可
